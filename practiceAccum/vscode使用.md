# 基本介绍
vscode 是一款 Miscroft 推出的非常好用的

### 无需插件完成

| 快捷键 | 功能 | 用处 |
|---|--|--|
| `ctrl` + `k` + `z` | 禅模式 | 全屏显示当前所有窗口 |
| `ctrl` + `shift` + `d` | 调试 | 打开侧边栏的调试窗口 | 
| `Ctrl` + `Shift` + `[` | 折叠代码 |
